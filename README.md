# Controle_Modelo_DDD

Para levantar o front, entre na pasta Modelo.Spa.Ui pelo CMD, e digite "ng serve -o"

Ja com o projeto na sua maquina, aperte Ctrl + F5 para levantar a API, caso necessario, botao direito em cima da solution >> propriedades >>
Selecione a opcao Multiple Startup projects procure o Modelo.Application e deixe ele como start, salve, e em seguida aperte Ctrl + f5 para levantar a Api
